# se_project_spots

### Overview

This project is a responsive webpage made to adapt to different screen sizes. Desktop, tablet and mobile devices.

**Intro**
Technologies used:
-HTML
-CSS
-CSS Grid
-media queries
-web fonts
-normalize.css
-figma

Techniques used:
-BEM Methodology
responsive design
semantic HTML
CSS file organization
font fallbacks
